# Balancing-robot
Arduino balancing robot sketch

Connect bluetooth to the robot
9600, 8n1

Commands:
if (cmd == '0') - START
if (cmd == 'X') - STOP
if (cmd == 'F') - Forward
if (cmd == 'B') - Backward
if (cmd == 'L') - turn Left
if (cmd == 'R') - turn Right
if (cmd == 'W') - spin right
if (cmd == 'Z') - spin left

All the letters are CAPITAL
